### Run Cypress

Clone repo down, and run cypress

###### Using npx

Note: npx is included with npm > v5.2 or can be installed separately.

`npx cypress open`

or by using yarn

`yarn run cypress open`

or the long way using the full path

`./node_modules/.bin/cypress open`
